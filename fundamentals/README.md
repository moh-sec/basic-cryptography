# Fundamentals of Cryptography

This folder contains the **core concepts of cryptography**.  
It is designed to help beginners understand the basic ideas before working with code.

## Contents
- `concepts.md` : Introduction to key concepts such as plaintext, ciphertext, keys, encryption, and hashing.  
- Additional theory files may be added in the future.

## How to Use
1. Start by reading `concepts.md` to understand the basic concepts.  
2. Afterward, explore the `classical` or `modern` folders for practical examples.
